# AI Study Buddy

An AI-powered study assistant built with Flowise and Google Gemini. It explains concepts step by step in a friendly, student-focused manner.

## Features
- Natural language question answering
- Step-by-step explanations
- Built visually using Flowise

## Tech Stack
- Flowise
- Google Gemini API

## How to Use
1. Import the provided Flowise JSON into Flowise.
2. Add your own Gemini API key.
3. Start chatting with the Study Buddy!
